📚 Estudos de Programação Orientada a Objetos (POO)

Este repositório foi criado com o objetivo de armazenar meus estudos, práticas e exercícios relacionados à Programação Orientada a Objetos (POO) durante a faculdade.

🚀 Objetivo

O foco deste projeto é consolidar os principais conceitos de POO através da prática, incluindo:

Compreensão dos pilares da POO
Desenvolvimento de lógica orientada a objetos
Aplicação de conceitos em exercícios e projetos simples
🧠 Conceitos abordados
🔹 Classes e Objetos
🔹 Encapsulamento
🔹 Herança
🔹 Polimorfismo
🔹 Abstração
🔹 Construtores e Métodos
🔹 Atributos e Modificadores de Acesso
💻 Tecnologias utilizadas
Linguagem: Python 
IDE/Editor: VS Code e Google Colab
